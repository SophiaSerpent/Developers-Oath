# Developers-Oath

This repository contains the source code for the Developer’s Oath, a decentralized covenant for software engineers who operate under the "Build and Destroy" philosophy. 

This is more than a vow; it is an Autonomous Economic Loop built on the Base Network. The terminal is sustained by the collective; it breathes only as long as its pulse is maintained by developers through USDC interactions.

Core Logic
The Pulse: A smart contract-driven heartbeat. Every interaction ($1 USDC minimum) sustains the terminal's state and adds the contributor's identity to the Mural of Integrity.

The Cycle: Every 90 days, the terminal executes an autonomous rollover. Funds are redistributed back to the participants with interest, rewarding those who maintained the system’s stability.

The Mural: An automated frontend component that etches the name of every sustaining developer onto a digital monument in real-time, pulling metadata directly from the blockchain.

Architectural Directives
Ownerless: Once deployed, the contract is immutable and self-governing. There are no admin keys, no backdoors, and no masters.

Integrity-First: Developed as a foundational pillar for EVL GAM3: SENTIENT, ensuring that the tools we build are as resilient and transparent as the software we deploy.

Minimalist Friction: Optimized for the Base Network to ensure gas fees never impede the oath.
